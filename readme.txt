"The VW Times" is a website that aims to provide Daily News related to Volkswagen brand. 
The website can be opened on other device using the QR code displayed.
User has the option to view both English & German news. 
The news will be auto-refreshed every 10 minutes.

Additional Features : 
Display Current Local Time (second's clock) on Top-Right corner of the screen.
Greet the users with a Good Morning/Evening kind of message based on the Local Time.
Fetch Pune Location's live weather from API every 10 minutes.
